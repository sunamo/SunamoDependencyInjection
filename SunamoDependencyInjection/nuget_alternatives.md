# NuGet Alternatives to SunamoDependencyInjection

This document lists popular NuGet packages that provide similar functionality to SunamoDependencyInjection.

## Overview

Dependency injection utilities

## Alternative Packages

### Microsoft.Extensions.DependencyInjection
- **NuGet**: Microsoft.Extensions.DependencyInjection
- **Purpose**: Official .NET DI container
- **Key Features**: Service lifetimes, scoping, IServiceProvider

### Autofac
- **NuGet**: Autofac
- **Purpose**: Popular IoC container
- **Key Features**: Modules, decorators, interceptors, lifetime scopes

### Ninject
- **NuGet**: Ninject
- **Purpose**: Lightweight DI framework
- **Key Features**: Convention-based binding, kernel configuration

### SimpleInjector
- **NuGet**: SimpleInjector
- **Purpose**: Fast DI container
- **Key Features**: High performance, verification, diagnostics

### DryIoc
- **NuGet**: DryIoc
- **Purpose**: Fast IoC container
- **Key Features**: Code generation, performance optimized

## Comparison Notes

Microsoft.Extensions.DI is the standard. Autofac for advanced features. SimpleInjector/DryIoc for performance.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **Microsoft.Extensions.DependencyInjection**: Official .NET DI container
- **Autofac**: Popular IoC container
- **Ninject**: Lightweight DI framework
